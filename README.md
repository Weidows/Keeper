# Intro

- ## 一个基于 `GitHub-Action` 的定时任务.
- ## 一个每日溜圈的`爬虫`.
  - ### 每日访问唤醒 `LeanCloud` 评论后台.
  - ### 每日获取`必应壁纸`.

---

# Feature

- 在指定时间对 `GitHub` 进行 `push` 以保持 Profile 格子绿油油.
- 每次执行任务会把 Log 记录在 `Autogreen.log` 中.
- 必应壁纸存储在[Bing](./Bing/)里面.
- 任务都在`daily.yml`里面,可自定义修改.
